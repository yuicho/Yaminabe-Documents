# 「闇鍋クラフト」参加同意事項
本同意事項は、「闇鍋クラフト」（以下「本サーバー」といいます）への参加、ならびに本サーバーに付随する Discord サーバー、ボイスチャット、Webサービス、イベント、配信、録画、画像投稿その他の関連サービスの利用に適用されます。
本サーバーへ参加し、🆗リアクションを行った時点で、参加者は本同意事項に同意したものとみなします。

## 第1条　適用対象
本同意事項は、以下のすべてに適用されます。
1. 本サーバー（Minecraft サーバー）
2. 本サーバー運営が管理する Discord サーバー
3. 本サーバーに関連するボイスチャット、通話、Webページ、フォーム、問い合わせ窓口
4. 本サーバーに関連する配信、録画、アーカイブ、切り抜き、スクリーンショット、告知画像、SNS投稿

## 第2条　参加条件
参加者は、以下の条件を満たすものとします。
1. 運営承認を受けていること
2. Discord その他運営が指定する連絡手段を利用できること
3. 本同意事項および別途定めるルールを遵守できること
4. 外部サービスの利用条件を満たしていること
5. 虚偽の申告を行わないこと

## 第3条　配信・録画・公開に関する同意
​参加者は、以下の事項にあらかじめ同意するものとします。
1. 本サーバー内でのプレイ状況が、他参加者または運営による配信、録画、アーカイブ、切り抜き、スクリーンショット、SNS投稿等に映り込む場合があること
2. 参加者の以下の情報が、前項の公開物に含まれる場合があること
  - Minecraft 上のプレイヤー名
  - スキン、見た目、所持品、建築物、発言内容
  - Discord 表示名または通話参加状況
  - ボイスチャット利用時の音声
3. 本サーバーでは、ボイスチャットの常時録音は行いません
4. 参加者が自ら行う配信、録画、アーカイブ公開および切り抜き公開については、当該配信等を行った本人の判断によるものとします
5. 参加者は、自身の映り込み、発言、音声その他前二項に定める内容が配信、録画、アーカイブ、切り抜き等に含まれることについて、参加時点で同意したものとみなします
6. 参加者は、自身の映り込み、発言または音声が含まれていることのみを理由として、当該配信、録画、アーカイブまたは切り抜きの削除、編集、非公開その他の対応を求めることはできません
7. 前各項の公開物は、サーバー紹介、活動報告、募集告知、配信アーカイブ、切り抜き、広報目的でも利用される場合があります

## 第4条　取得情報および利用目的
運営は、本サーバーの運営に必要な範囲で、以下の情報を取得する場合があります。
1. Minecraft アカウント名、UUID
2. Discord アカウント情報
3. 接続記録、IPアドレス、参加申請情報
4. チャットログ、問い合わせ内容、通報内容
5. ボイスチャットの接続状況その他障害調査に必要な情報
6. 運営対応の記録、警告歴、制限措置履歴
これらの情報は、以下の目的で利用します。
1. 参加審査、本人確認、権限管理
2. サーバー運営、保守、障害調査
3. 荒らし・迷惑行為・不正行為の防止および対応
4. トラブル対応、通報対応、再発防止
5. コミュニティの安全確保
6. 必要な運営連絡

## 第5条　ログ・証跡の保存
1. 運営は、前条の目的のため、接続ログ、チャットログ、通報対応記録、処分履歴等を保存することがあります
2. 保存期間は、運営上必要な範囲で合理的に定めるものとします
3. 保存された情報は、運営メンバーのうち必要な者のみが閲覧できるものとします
4. 法令に基づく場合その他正当な理由がある場合を除き、運営はこれらの情報を目的外に利用しません

## 第6条　禁止事項
参加者は、以下の行為をしてはなりません。
1. 荒らし、迷惑行為、嫌がらせ、つきまとい、暴言、脅迫、差別的言動
2. 他者の個人情報、私的情報、未公開情報を本人の許可なく共有、暴露、詮索する行為
3. なりすまし、虚偽申告、誤解を招く表示
4. チート、グリッチ悪用、不正ツール利用、運営の想定しない不正行為
5. サーバー、他参加者、外部サービスに過度な負荷または損害を与える行為
6. 運営判断に反する宣伝、勧誘、政治活動、宗教活動、営業活動
7. 他参加者の配信活動を妨害する行為
8. 本サーバー外であっても、本サーバー参加者に対する継続的な嫌がらせその他運営が不適切と判断する行為
9. 法令、公序良俗、各外部サービスの規約または運営ルールに違反する行為

## ​第7条　プライバシーおよび個人情報
1. ​参加者は、他参加者の個人情報、連絡先、居住地、勤務先、学校、顔写真、家族情報、未公開アカウント、DM内容その他私的情報を、本人の明示的な許可なく取得・保存・共有・公開してはなりません
2. 通話内容、個別相談、運営対応の内容を、関係者の許可なく外部へ転載してはなりません
   ただし、第3条に基づき参加者自身が行う配信、録画、アーカイブ、切り抜きについてはこの限りではありません
3. 配信や公開の過程で他参加者の情報が映り込む場合には、参加者は必要な配慮を行うものとします

## 第8条　参加者の制作物等
1. 参加者が本サーバー内で作成した建築、企画、画像、文章その他の制作物に関する権利は、原則として作成者に帰属します。
2. 参加者が本サーバー内または本サーバー運営が管理するDiscordサーバー内に投稿・共有した、スクリーンショット、画像、文章、企画内容その他の素材（以下「投稿素材」といいます）について、運営は、本サーバーの紹介、活動報告、募集告知、Webページ掲載、SNS投稿、告知画像作成その他本サーバーの運営・広報に必要な範囲で、無償で利用できるものとします。
3. 運営は、投稿素材の利用にあたり、必要に応じてトリミング、サイズ調整、文字入れ、抜粋その他、趣旨を損なわない範囲の軽微な編集を行うことがあります。
4. 前二項は、参加者が自ら制作・公開した配信、録画、アーカイブ、切り抜き、動画、音声コンテンツそのものを、運営が無断で転載、再配信、編集または二次利用することを許諾するものではありません。ただし、これらが第3条に基づく公開物に付随的に映り込む場合、または参加者本人が投稿素材として共有した場合はこの限りではありません。

## 第9条　外部サービスの利用
1. 本サーバーは Minecraft、Discord その他外部サービスを利用して運営されます
2. 参加者は、これら外部サービスの利用規約、ガイドライン、年齢要件等を自ら確認し、遵守するものとします
3. 外部サービスの障害、仕様変更、規約変更、アカウント停止等により本サーバーの一部機能が利用できなくなる場合があります
4. これにより参加者に生じた不利益について、運営に故意または重過失がある場合を除き、運営は責任を負いません

## 第10条　違反時の措置
運営は、参加者が本同意事項または別途定めるルールに違反したと判断した場合、事前通知なく以下の措置を行うことがあります。
1. 注意、警告
2. 発言制限、通話制限、権限制限
3. 一時停止
4. 追放、BAN、招待取消
5. 必要に応じた証跡保全、関係者への共有、法令に基づく対応
運営は、コミュニティの安全確保のため、緊急性がある場合には先行して制限措置を行えるものとします。

## 第11条　免責
1. 参加者間のトラブルについて、運営は必要な範囲で対応を行いますが、完全な解決を保証するものではありません
2. 本サーバーは、継続的な稼働、完全な安全性、データの完全保存を保証するものではありません
3. 天災、障害、外部サービス停止、通信障害、アカウント停止、第三者行為その他運営の合理的支配を超える事情により生じた損害について、運営は責任を負いません

## 第12条　変更
1. 運営は、本サーバーの運営、安全確保、外部サービスの仕様変更、法令・規約への対応、荒らし・迷惑行為・不正行為への対策その他必要な場合に、本同意事項を変更することがあります。
2. 運営は、本同意事項を変更する場合、変更内容、変更理由および効力発生日を、Discordその他運営が適切と判断する方法で周知します。
3. 誤字脱字の修正、表記の整理、条番号の修正その他参加者の権利義務に実質的な影響を与えない変更は、周知後すぐに適用できるものとします。
4. 前項以外の変更は、原則として周知から7日後に適用します。
5. 参加者の権利義務に実質的な影響を与える重大な変更は、原則として周知から14日後に適用し、運営内で協議のうえ行います。
6. 緊急性がある場合、またはコミュニティの安全確保、個人情報保護、法令・外部サービス規約への対応のため速やかな変更が必要な場合は、運営責任者または対応可能な運営メンバーの判断により、暫定的に即時変更できるものとします。この場合、運営は速やかに変更内容を周知し、必要に応じて運営内で見直します。
7. 変更後の効力発生日以降も本サーバーへの参加を継続した場合、参加者は変更後の内容に同意したものとみなします。
